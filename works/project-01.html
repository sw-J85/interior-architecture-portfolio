Project 01

Co-working Space Design
Educational / Evaluation Project

Overview

  • Type · Co-working Space
  • Role · Planning · Spatial Design
  • Scope · Interior Architecture
  • Tools · AutoCAD · SketchUp · 3ds Max

Context & Requirement

본 프로젝트는 다수의 사용자가 동시에 사용하는 코워킹 공간을 대상으로 한다. 기능 간 충돌을 최소화하고, 작업과 휴식의 경계를 명확히 구분하는 것이 주요 과제였다.

Concept

공간은 기능 단위로 분해한 뒤, 사용자 흐름을 기준으로 재구성하였다. 과도한 연출보다 구조적 안정성과 반복 사용에 적합한 구성을 우선하였다.

Spatial Structure

주요 기능 영역을 최소 단위로 분해하고 관계를 설정하였다.

Design Decision

  • 동선 교차를 최소화하는 배치
  • 유지관리가 용이한 마감재 우선 선택
  • 도면으로 명확히 설명 가능한 구성만 채택

Rendering

Result

본 프로젝트를 통해 공간 기능을 구조적으로 정리하는 설계 접근과 판단 기준의 중요성을 확인하였다.
